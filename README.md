# G-Wallet

Учебный проект 1 курса по дисциплине **«Программирование на Python»**.

## Описание проекта

Проект представляет собой mock-криптокошелёк на Django: пользователь может зарегистрироваться, войти в аккаунт, просматривать цены криптовалют, покупать и продавать токены за виртуальные деньги, а также видеть состояние своего баланса.  
Приложение реализовано как веб-сервис на **Django** с использованием HTML-шаблонов, CSS и JavaScript. Данные пользователя, профиль и купленные токены хранятся в базе данных SQLite через стандартную ORM Django.  
Проект не является настоящим криптокошельком и не выполняет реальные финансовые операции. Все покупки и продажи происходят на моковом балансе пользователя, который создаётся внутри приложения.

## Установка и запуск
### Вариант 1. Запуск через Docker

Требования:
- Docker;
- Docker Compose.

Клонируйте репозиторий:
```bash
git clone https://github.com/repeatuntil/g_wallet.git
cd g_wallet
```

Соберите и запустите контейнер:
```bash
docker compose up --build
```

После запуска приложение будет доступно по адресу:
```text
http://127.0.0.1:8000/
```

### Вариант 2. Локальный запуск

Требования:
- python 3.10 и выше

Клонируйте репозиторий:
```bash
git clone https://github.com/repeatuntil/g_wallet.git
cd g_wallet
```

Создайте виртуальное окружение:
```bash
python -m venv venv
```

Активируйте окружение.  

Для Linux/macOS:
```bash
source venv/bin/activate
```

Для Windows PowerShell:
```powershell
.\venv\Scripts\Activate.ps1
```

Установите зависимости:
```bash
pip install -r requirements.txt
```

Примените миграции базы данных:
```bash
python manage.py migrate
```

Запустите сервер разработки:
```bash
python manage.py runserver 127.0.0.1:8000
```

Откройте приложение в браузере:
```text
http://127.0.0.1:8000/
```

## Примечания
- Для получения цен и построения графиков требуется подключение к интернету.
- Покупка и продажа валют выполняются только внутри приложения и не связаны с реальными криптовалютными транзакциями.
